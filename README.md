### Quick Setup (GitHub Pages)

1. **Push your code to GitHub** (if you haven't already):
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Enable GitHub Pages**:
   - Go to your repository on GitHub: `https://github.com/OmarMusaev/site`
   - Click on **Settings** â†’ **Pages**
   - Under **Source**, select **GitHub Actions**
   - Save the settings

3. **Your site will be live at**:
   ```
   https://omarMusaev.github.io/site/
   ```
   (Note: It may take a few minutes for the first deployment to complete)

### Alternative: Quick Deploy with Netlify

If you prefer a different hosting option:

1. Go to [netlify.com](https://www.netlify.com/)
2. Sign up/login
3. Drag and drop your `Site` folder onto Netlify
4. Get an instant live link!
